# EcoModa | Tienda Ropa Sostenible

Proyecto académico desarrollado por Aída Guadalupe Meza Gómez (2025)  
Materia: Proyecto II  
Modalidad: Desarrollo con React y muestra en HTML + CSS  

---

##  Descripción del proyecto

EcoModa es una propuesta digital que promueve el consumo responsable y la moda sostenible.  
El sitio web presenta información, artículos y videos sobre prácticas ecológicas en la industria textil.

> "Moda ecológica para un mundo mejor"
